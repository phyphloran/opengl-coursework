# opengl-coursework

Курсовой проект на классическом OpenGL: визуализация тора, цилиндра и линии их пересечения методом пересечения плоскостей.

## Идея проекта

В сцене показаны:
- зеленый тор;
- красный цилиндр;
- желтая пространственная кривая пересечения этих поверхностей;
- набор вспомогательных секущих плоскостей `y = const`, которые иллюстрируют метод построения.

Цилиндр смещен к внешнему краю тора, чтобы композиция была ближе к референсному изображению из задания.

## Что реализовано

- Отрисовка тора вручную через `GL_QUAD_STRIP`.
- Отрисовка цилиндра и его крышек через `GLUquadric`.
- Черный фон.
- Перспективная проекция с корректным `aspect ratio` при изменении размеров окна.
- Вращение камеры мышью.
- Переключение вспомогательных секущих плоскостей клавишей `E`.
- Освещение сцены с мягкими тенями.
- Построение линии пересечения для текущего смещенного положения цилиндра.

## Метод пересечения плоскостей

В проекте используется семейство горизонтальных плоскостей вида `y = const`.

Для каждой такой плоскости:
- цилиндр пересекается по двум прямым в сечении;
- тор пересекается по окружности или паре окружностей;
- точки пересечения этих сечений дают точки искомой пространственной линии.

Именно эта идея используется и для визуализации вспомогательных построений, и для вычисления желтой кривой пересечения.

## Управление

- `ЛКМ` + движение мыши: вращение камеры вокруг сцены.
- `E`: скрыть или показать вспомогательные плоскости.

## Сборка

Проект рассчитан на Windows и Visual Studio 2022.

Собрать можно так:
1. Открыть `Prime.sln` в Visual Studio.
2. Выбрать конфигурацию `Debug | x64` или `Release | x64`.
3. Выполнить сборку проекта.

Результат сборки:
- `x64\Debug\Prime.exe`

## Технические детали

- Используются `opengl32.lib` и `glu32.lib`.
- Окно создается через WinAPI.
- OpenGL-контекст создается через `wglCreateContext`.
- Камера реализована как orbit-camera вокруг центра сцены.
