# opengl-coursework

Курсовая OpenGL-сцена: **цилиндр и тор, оси пересекаются под прямым углом**.

Визуализация построена по заданию на фото:

- зелёный тор;
- красный цилиндр;
- оси поверхностей пересекаются под прямым углом;
- жёлтая линия показывает линию пересечения поверхностей;
- серые каркасные сферы иллюстрируют метод концентрических сфер с общим центром в точке пересечения осей.

Проект очищен от пользовательских файлов Visual Studio, временных файлов и результатов сборки. В репозитории остаются только исходники и файлы проекта.
